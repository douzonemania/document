## 사장님 DB
### ceo : 고객(사장님)
- 사장님마다 쇼핑몰 DB 생성(고객 id로 DB 구분)
### agreement : 약관
- 약관은 default로 주어지도록
- 사장님이 변경 가능(update)
### main_menu : 메인메뉴
- header
- shop, new, best, review, faq, sale +?
- 주어진 메뉴들 중, 최대 5개까지 선택 가능하도록
### sub_menu : 서브메뉴
- 기본으로 main 메뉴 포함
- 사장님이 추가한 메뉴들을 drag&drop할 수 있음(index)
### custom : 커스텀
- 커스텀 할 수 있는 메뉴는 최대 5개
### contents : 컨텐츠
- - -
